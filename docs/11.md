## 问题

**问题 2：** 以下哪一项是 _而不是_ 的有效评论：

一个。 `/** comment */`
b。 `/* comment */`
c。 `/* comment`
d。 `// comment`

**问题 3** ：如果在运行时看到以下错误，应该检查的第一件事是：

```

HelloWorldApp.java.

```
